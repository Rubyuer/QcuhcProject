# 球琛体育——足球预约好助手

    "球琛体育——足球预约好助手"该应用是一款线上预约足球场，和学习足球技巧的app。
        该应用通过线上选择足球馆，添加预定场馆数量，实现线上预定足球馆的应用。
        同是查看关于足球的国内外的相关新闻介绍，帮助您了解足球相关资讯。
        "宝典"实现学习足球技巧，包括足球基础知识、足球颠球技术、足球射门、足球技巧和足球射门触球部位图解等类型。

        意在给喜欢足球项目的人，提供一个现实预约足球场馆的平台。
